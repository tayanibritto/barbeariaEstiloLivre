Título: Barbearia Estilo Livre (empresa fictícia)

Descrição: Página web de uma barbearia fictícia chamada "Estilo Livre", em que foi utilizado o framework Bootstrap. Objetivo: criar um site moderno, responsivo e visualmente atrativo, utilizando os componentes pré-construídos do Bootstrap. Foram realizadas algumas personalizações nos temas do Bootstrap, para que a página não ficasse genérica.

Tecnologias utilizadas: HTML5, Bootstrap

Pré-requisitos: Navegador atualizado (projeto foi testado e funciona no Google Chrome Versão 142.0.7444.163 64 bits - Caso utilize este navegador, recomenda-se tal versão ou superior).

Instalação:

Passos para clonar e rodar: git clone https://github.com/tayanibritto/barbeariaEstiloLivre.git / cd barbeariaEstiloLivre / abrir index.html no navegador Como usar: Para testar a responsividade da página, você pode utilizar o DevTools do Navegador e escolher os vários tipos de dispositivos. Observações: A estrutura do layout responsivo funciona 100%, porém a configuração JavaScript (ou a linguagem que preferir) do formulário de contato precisarão da configuração do desenvolvedor. As informações personalizadas (como imagens, depoimentos e dados da empresa) também precisarão de alteração. Este projeto utiliza Bootstrap para estilização, mas os arquivos CSS já estão compilados. Não é necessário instalar dependências ou compilar o SCSS/Bootstrap para executar.
